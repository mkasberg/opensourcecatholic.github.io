Catholic Diocese App for iOS/Android powers new Jesuit App

Jun 30, 2012 • Jeff Geerling

Ever since building Catholic STL, an iOS app for the Archdiocese of St. Louis, and then generalizing the code to produce the Catholic Diocese App for iOS, I’ve wanted to develop a version for Android.

    

A few months ago, the Jesuit Conference contacted me, and finally gave me the ability to do this. Not only was I able to build the Jesuit App for iOS and Android phones (Android version will be on Google Play soon!), but I was able to generalize the code from the Android version to post the Catholic Diocese App for Android.

I’ll be cleaning up some documentation here and there in the coming months, and I invite any other developers interested in helping to do so—all of the source code and documentation is available on GitHub (links below), and you can follow along with the latest news on the project page: Catholic Diocese App.

Grab the source and learn more on GitHub:

  • Catholic Diocese App - iOS
  • Catholic Diocese App - Android

You can download the Jesuit App from the iTunes store (for iOS) or Google Play (for Android—coming soon!). Read the Jesuit Conference’s announcement.
